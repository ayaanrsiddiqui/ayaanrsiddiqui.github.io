# ayaansiddiqui.github.io

This is the repository for my professional portfolio and personal website, hosted via GitHub Pages.

## 🚀 Tech Stack
- **HTML5 / CSS3**: Minimalist, responsive design.
- **GitHub Pages**: Automated deployment pipeline.
- **fnm / Node.js**: Local development environment management.

## 🛠️ Local Development
To make changes to this site locally:
1. Clone the repository:
   `git clone https://github.com/ayaanrsiddiqui/ayaanrsiddiqui.github.io.git`
2. Open `index.html` in your browser or use a live server extension in VS Code/Cursor.
